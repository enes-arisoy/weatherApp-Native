# 🌤️ WeatherApp

A beautiful and intuitive weather application built with React Native and Expo. Get real-time weather updates, forecasts, and more! ☀️🌧️❄️

## ✨ Features

- 🔍 **Search Locations**: Easily search for any city worldwide
- 📍 **Current Weather**: Get real-time temperature, humidity, wind speed, and conditions
- 📅 **7-Day Forecast**: View daily weather predictions
- 🎨 **Beautiful UI**: Modern design with gradient backgrounds and smooth animations
- 📱 **Cross-Platform**: Works on iOS and Android via Expo

## 📖 Usage

1. Open the app 📱
2. Tap the search icon 🔍 to find a location
3. Enter a city name and select from suggestions
4. View current weather and 7-day forecast 🌟

## 🛠️ Technologies Used

- **React Native** - Cross-platform mobile development
- **Expo** - Framework for universal React applications
- **NativeWind** - Tailwind CSS for React Native
- **Axios** - HTTP client for API calls
- **WeatherAPI** - Weather data provider
